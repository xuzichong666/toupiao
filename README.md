# 投票选择微信小程序

一个原生微信小程序 MVP，用于创建投票、匿名参与投票，并在投票后查看结果。

## 功能

- 创建单选或多选投票
- 添加 2 到 10 个选项
- 设置匿名投票、截止时间和多选数量
- 创建成功后分享邀请
- 本地匿名投票
- 同一设备防重复投票
- 实时查看票数、占比和最高票选项
- 首页按全部、我发起的、我参与的、已结束筛选
- 支持结束、取消、删除投票
- 数据保存在微信小程序本地存储中

## 运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择当前目录：`/home/xuzichong/project`。
4. AppID 可使用测试号或游客模式。
5. 编译后从首页创建投票并测试流程。

## 说明

当前版本不接后端或微信云开发，投票数据只保存在当前设备，适合演示和功能验证。
